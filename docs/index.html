A) HTML Structure - top

This theme is a responsive layout using twitter bootstrap 2.0 (All bootrap features are included in this theme so you can add or adjust any features or styles found in the bootstrap documentation). All of the content is held on the one page (index.html) and the width of the content area adjusts for mobile, and 980px and 1200px wide.

Each section is wrapped up in a few divs to give some CSS glow effects, etc. If you would like to edit the color, font, you would do the following:

	#primaryContent a {
		color: #someColor;
	}

If you find that your new style is not overriding, it is most likely because of a specificity problem. Scroll down in your CSS file and make sure that there isn't a similar style that has more weight.

I.E.

	#wrap #primaryContent a {
		color: #someColor;
	}

So, to ensure that your new styles are applied, make sure that they carry enough "weight" and that there isn't a style lower in the CSS file that is being applied after yours.

B) CSS Files and Structure - top

I'm using two CSS files in this theme. The first one is twitter bootstrap 2.0 and you can use anything you find in there documentation. Keep in mind, that these values might be overridden somewhere else in the file and you should make any customization in the site.css file, and not editing the bootstrap files.

The second file (site.css) contains all of the specific stylings for the page. The file is separated into sections and each section has media queries for the different screen sizes. When your customizing this, be sure to make appropriate changes in each media query, and test that you get your desired changes at those specific screen sizes.

	/* Global Style */

	some code

	@media {
	
	some code
	
	}
	
	etc, etc.

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

Note: Some pieces of content are turned off at smaller screen sizes. Inside the media queries for smaller screens you will see 'display: none;' set on those elements. To turn them on, just set to 'display: block;'.

C) JavaScript - top

This theme imports several Javascript files.

  1. Google Analytics
  2. jQuery
  3. jQuery Easings
  4. Modernizr
  5. Twitter Bootstrap
  1. Google Analytics is a free site tracking system from Google - Just add your unique ID - If your not going to use Google Analytics removed this script.
  2. jQuery is a Javascript library that greatly reduces the amount of code that you must write.
  3. jQuery Easings adds a more natural effect to jQuery animations. This is used when sizing the Globules.
  4. Modernizr is used here to help standardize the way some our CSS3 items work accross browsers.
  5. Twitter Bootstrap JS adds in functionality for all the components you will find in there documentation. We specifically use the Carousel and Tooltips. If you wish to lighten the page size, you can download a customized package of twitter bootstrap 2.0 and include only the components you wish to use.
